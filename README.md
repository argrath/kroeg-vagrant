# Vagrantfile for [Kroeg](https://github.com/puckipedia/Kroeg)

## How to use

 1. $ `vagrant up`
 1. $ `vagrant ssh`
 1. $ (In the guest) `sh startup.sh`
 1. Access http://192.168.1.203:5000 (by default)

You can change IP address by editing $ipaddress on Vagrantfile.
